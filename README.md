# ARGB-WLED-PC-Controller
A device for controlling ARGB lights via an ESP32 running WLED. Originally intentioned for desktop PC components like fans and LED strips using 3-pin ARGB connectors.

## Initial Devboard Design
In `/Devboard` you can find the V1 perfboard design schematics and images. This design has a few flaws outlined in its README, but it does work! Below is an image of the board and its enclosure.
